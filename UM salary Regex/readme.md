UM 2012 Salary Information
----------------------------------
**UM 2012 salary Regex.ipynb** uses regular expression to query the University of Michigan employee salary information (salaries2012.txt).

1) Gather the number of employees on each campus (Ann Arbor, Dearnborn, Flint)

2) Sort the job titles by the number of people employed under that job title

3) Sort the job titles by the average salary for that title

4) Create a word cloud of the First names for professors in Ann Arbor
